## Como usar este repositório

1. Atualizar `progress.md` toda sexta-feira.
2. Atualizar `pmle-domain-checklist.md` conforme o avanço.
3. Criar cada projeto como repositório separado usando `templates/project-template`.
4. Rodar `python scripts/pre_publish_check.py --repo-root .` antes de publicar.
5. Fixar este repositório e os três projetos principais no perfil do GitHub.
