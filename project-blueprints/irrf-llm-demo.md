# Blueprint — IRRF LLM Demo

**Objetivo:** demonstrar LLM + regras determinísticas, saída estruturada, cache e observabilidade.

**Dataset seguro:** descrições e documentos completamente sintéticos.

**Stack:** Python, FastAPI, Pydantic, Docker, Cloud Run e BigQuery.

**Métricas:** macro-F1, exact match, fallback rate, cache hit-rate, custo estimado e p95 de latência.

**Regra de PI:** não reutilizar dados, regras fiscais internas, prompts empresariais ou código de terceiros.
