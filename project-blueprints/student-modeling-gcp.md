# Blueprint — Student Modeling on GCP

**Objetivo:** demonstrar ML clássico, BigQuery ML, avaliação, API e monitoramento.

**Dataset seguro:** OULAD ou UCI Student Performance.

**Stack:** BigQuery, BigQuery ML, Python, FastAPI, Docker e Cloud Run.

**Métricas:** AUC-ROC, PR-AUC, F1, calibração e slice metrics.

**Regra de PI:** não usar dados, código ou consultas da tese. Reimplementar com dados públicos.
