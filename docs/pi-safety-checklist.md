# Checklist de publicação sem risco de PI

- [ ] O código foi escrito do zero para o portfólio.
- [ ] Os dados são sintéticos ou públicos com licença compatível.
- [ ] Não existem nomes de clientes, parceiros, estudantes ou instituições vinculados a casos protegidos.
- [ ] Não existem documentos internos, prints, tabelas reais ou prompts internos.
- [ ] Não existem tokens, chaves, service accounts ou arquivos `.env`.
- [ ] O histórico Git foi revisado.
- [ ] Outputs de notebooks foram removidos.
- [ ] Metadados de imagens foram revisados.
- [ ] `DISCLAIMER.md` foi preenchido.
- [ ] NIT/PROPI foi consultado quando houver dúvida.
